# 英语5个语法层次:

* 句子

* 分句

* 词组

* 词

* 词素：最小音义结合单位

  * 词根：单词主要含义
    * 自由词根：本身是独立单词
    * 粘附词根：本身不是独立单词
  * 词缀：单词次要含义
    * 前缀：词义变化
    * 后缀：词性变化

  

单词不一定等于词根与词缀的简单拼接

1. 字母的增加
2. 字母的脱落
3. 字母的变更
4. 连字符的使用

# 英语来源
