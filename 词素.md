构词法中经常用元音i、o、u作为连接字母，不表示意思，起连接作用。连接字母：u，i，o。

# 词根

| 词根 | 释义 | 引申 | 备注 |
| :- | :- | :- | :- |
| ac(u)-, acerb- | 尖 | 酸、锐利 |
| act | 行动、做 |
| aer(o) | 空气、充气 | | 由单词air元音音变（i->e）意思不变 |
| ag | 行动、做 | | 由act发生辅音音变(ct-g)意思不变 |
| agog | 引导 |
| agon | 挣扎、斗争 |
| agr | 田地、农业 |
| alt | 高 | 由tal转化而成 |
| alter | 改变 | alt(高)+er(....人) |
| ambul，amb | 走 |
| anim | 生命、精神 | 由单词animal演变 |
| ann，enn | 年、一年 | enn由ann元音音变(a->e)意思不变 |
| log | 学科 |
| trans | 通过 |
